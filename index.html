프로젝트 요약

  • Spring Security 기반의 세션 인증과 권한 분리(사용자/관리자).
  • 게시글·스토리·공지 CRUD와 검색/페이징/좋아요/댓글.
  • 인라인 이미지 업로드, 임시파일 자동 정리, 다운로드 지원.
  • Thymeleaf 템플릿과 정적 리소스 공개 경로 관리.
  • 홈 대시보드에 인기/최신, 방문자 집계, 태그/소개/히어로 이미지 표시.

상단 네비게이션을 이용해 각 기능의 상세를 확인하세요.

핵심 기술 스택

Spring Boot 3 · Security · Thymeleaf · JPA · AWS S3 업로드 구성과 대시보드용 집계 작업을 한 곳에 정리했습니다.
